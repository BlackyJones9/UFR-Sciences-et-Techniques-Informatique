Algorithme PPP en Langage Algorithmique : 

Structure Point
	Tableau[1,2] de réels compris entre 0 et 1

La fonction Index renvoie l'indice de l'élément donné en argument dans une liste également donnée en argument
La fonction calculateDistance renvoie un réel correspondant à la longueur totale du circuit
La fonction Insérer insère un élément dans une liste à l'indice indiqué
La fonction Retirer retire un élément dans une liste étant à un certain indice
La fonction Inverser inverse l'ordre d'une liste
La fonction calculateDistance calcule la distance totale du circuit selon une matrice de distance

Procédure PPP(longueurCircuit : Entier, circuit : Tableau[1..n+1], list : Tableau[1..n], D : Tableau[1..n] de Tableau[1..n] de réels):
	Var pointList : Tableau[1..n], i, j, index, Q1, minValue, minInsert, indexCycle, indexPoint, indexMin : Entier, res : Tableau[1..n+1], dist, distMin : Réels
	Début
	pointList <- list
	index <- 0
	Q1 <- pointList[0]
	Ajouter(res, Q1)
	RetirerIndex(pointList, index)
	Tant que (Longueur(pointList) != 0)
		Début
		minValue <- 0  #Représente l'indice du point le plus proche du circuit dans pointList
		minInsert <- 0  #Représente l'indice du point du circuit le plus proche de minValue
		Pour i allant de 1 à Longueur(pointList) faire
			Début
			Pour j allant de 1 à Longueur(res) faire
				Début
				indexCycle <- Index(list, res[j])
				indexPoint <- Index(list, pointList[i])  #Récupère les indices selon list (qui reste fixe) afin d'ajouter les bons points au circuit
				indexMin <- Index(list, pointList[minValue])
				dist <- D[indexCycle][indexPoint]
				distMin <- D[indexCycle][indexMin]
				Si distMin > dist faire #On cherche les indices minimums
					Début
					minValue <- i
					minInsert <- j
					FinSi
				FinPour
			FinPour
		Insérer(res, minInsert, pointList[minValue])  #Ajoute le point le plus proche au circuit
		Retirer(pointList, minValue)  #Le point a été ajouté donc il est retiré de pointList
		FinTantQue
	Inverser(res)  #On ajoute par insertion donc le circuit est inversé
	Ajouter(res, res[0])  #On ajoute le point de départ à la fin afin de créer un vrai circuit
	longueurCircuit <- calculateDistance(D, res, list)
	circuit <- res
	FinProcédure
	

La procédure décroiser applique un algorithme de décroisement sur le circuit donné selon une matrice distance, la distance de ce circuit et une liste de points de référence

Procédure OptPPP(circuit : Tableau[1..n+1], list : Tableau[1..n], D : Tableau[1..n] de Tableau[1..n] de réels):
	Var i, j : Entier, circuit : Tableau[1..n+1], dist : Réel
	Début
	PPP(dist, circuit, list, D)  #On récupère le résultat de l'algorithme PPP
	décroiser(D, dist, circuit, list)
